# DE Failure Risk Categorization

This repository contains the Python implementation associated with the paper:

**Risk Categorization and Threshold-Based Failure Prediction for the Differential Evolution Algorithm**

Authors: Amani Saad, Andries P. Engelbrecht, and Salman A. Khan

## Overview

The code implements the risk categorization and threshold-based failure prediction procedure for the Differential Evolution algorithm.

The program reads the required input data, applies the threshold-based analysis, assigns risk categories, and produces the corresponding output results.
